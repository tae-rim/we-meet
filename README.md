# we-meet

# 원하는 위치에서
git clone https://github.com/your-username/2025위밋.git
cd 2025위밋

### Frontend
```bash
cd frontend
npm install
npm start
```
http://localhost:3000에서 실행됩니다.

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### 환경변수 설정
```bash
cd backend
python -m venv venv
venv\Scripts\Activate
```
