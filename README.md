# we-meet

### 원하는 폴더 위치에서
```bash
git clone https://github.com/tae-rim/we-meet.git
```

### Frontend
```bash
cd we-meet/frontend
npm install
npm start
```
http://localhost:3000에서 실행됩니다.

### Backend
```bash
cd we-meet/backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### 환경변수 설정
```bash
cd backend
python -m venv venv
venv\Scripts\Activate
```
